# Product Strategy: Arogya Mitra AI 

## 1. The Problem (Pain Point)
In South India, many patients (especially in rural Telangana, Andhra, and Karnataka) receive handwritten prescriptions. Misunderstanding dosages or drug names leads to a 15% increase in medication errors. 

## 2. Target Persona
- **Primary:** Non-English speaking patients (ages 45-70) in Tier 2/3 cities.
- **Secondary:** Caregivers who need to explain medication to elderly parents in regional languages (Telugu, Kannada, Tamil, Hindi).

## 3. Product Roadmap
- **V1 (MVP):** OCR + Basic LLM translation.
- **V2 (Validation):** Integration with a drug-interaction database (safety layer).
- **V3 (Scale):** Voice-output for illiterate users (Agentic Audio).

## 4. AI Guardrails & Ethics
- **Safety First:** The app includes a mandatory disclaimer: "AI-generated content. Verify with a pharmacist."
- **Confidence Threshold:** If OCR confidence is <85%, the system prompts the user to "Retake Photo" rather than guessing.

## 5. Success Metrics (KPIs)
- **Accuracy:** Goal of >95% accuracy in regional language medical terminology.
- **Latency:** <3 seconds from photo upload to explanation.
